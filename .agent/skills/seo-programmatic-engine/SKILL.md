# SEO Programmatic Engine

## Target query patterns
- "how many batteries for {5kW|10kW} solar"
- "solar battery sizing calculator"
- "how many kWh battery do I need for outages"
- "{location} solar sun hours" (later)

## Page rules
- Each page must provide unique value: examples, tables, FAQs, assumptions.
- Use helpful engineer tone: simple, direct, non-salesy.

## Required page sections (calculator page)
- Calculator
- “Assumptions & defaults” panel
- “Common mistakes” (kW vs kWh)
- FAQ (short + practical)
- CTA: “Download your BatteryBlueprint PDF”

## Technical SEO
- Use Next.js metadata per route
- Use clean headings, schema FAQ if applicable
- No keyword stuffing
